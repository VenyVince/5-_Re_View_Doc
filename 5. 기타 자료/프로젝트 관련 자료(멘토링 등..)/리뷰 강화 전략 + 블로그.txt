🧭 웹 플랫폼에 최적화된 리뷰 중심 기능 강화 전략
1. 리뷰 중심 메인 페이지 구성
- 리뷰 피드 기반 홈 화면: 상품이 아닌 리뷰가 중심이 되는 홈 → “이번 주 인기 리뷰”, “피부MBTI별 추천 후기”, “신규 리뷰러”
- 탭 구성: 전체 리뷰, 피부MBTI별, 태그별, 운영자 추천, 베스트 리뷰 등으로 분류
- 리뷰 카드 UI: 이미지 + 요약 텍스트 + 좋아요 수 + 피부MBTI + 사용자명 표시

2. 검색 기능 강화
- 리뷰 중심 검색 탭: 상품 검색과 별도로 리뷰만 검색하는 탭 제공
- 피부MBTI 필터링: “DSNT 사용자 리뷰만 보기” 등 필터링 옵션
- 태그 기반 검색: “#보습강력”, “#민감피부극찬” 등 기능성 태그로 리뷰 탐색
- 정렬 옵션 다양화: 좋아요 순, 최신순, 피부MBTI 일치율 순 등

3. 리뷰 작성 유도 및 보상 강화
- 리뷰 미션 시스템: “이번 주 3개 제품 리뷰하면 추가 포인트 지급”
- 리뷰 작성 가이드: 피부 상태, 사용 기간, 사용 방법, 추천 여부 등 항목별 템플릿 제공
- 리뷰 작성 후 즉시 피드백: “리뷰가 등록되었어요! 3명이 공감했어요” → 즉각 반응으로 만족도 상승

4. 리뷰 신뢰도 및 공감 요소 강화
- 피부MBTI별 만족도 통계: “DSNT 사용자 중 87%가 만족” → 리뷰 신뢰도 상승
- 리뷰 요약 기능: AI 기반으로 리뷰 키워드 자동 추출 → “이 제품에 대한 리뷰 키워드: 촉촉함, 무향, 민감피부 적합”
- 공감 버튼 다양화: “나도 이랬어요”, “정보 유익해요”, “사진 도움돼요” 등 좋아요 외 공감 방식 추가

5. 리뷰 기반 추천 기능 강화
- 리뷰 기반 상품 추천: “당신과 피부MBTI가 같은 사람들이 많이 리뷰한 제품” 추천
- 리뷰 기반 검색어 자동완성: 예: “건조한 피부” 입력 시 → “건조한 피부에 좋았던 리뷰 많은 제품” 자동 제안

6. 커뮤니티 기능 확장
- 리뷰러 프로필 페이지: 리뷰 수, 좋아요 수, 피부MBTI, 활동 이력 등 표시
- 팔로우 기능: 리뷰러 팔로우 → 새 리뷰 알림
- 리뷰 댓글에 사용자 피부MBTI 표시: 공감도 높은 댓글 탐색 가능



✅ 후기 중심 플랫폼으로 강화할 수 있는 기능 제안
1. 리뷰 탐색 경험 강화
- 리뷰 전용 홈 탭: 상품 중심이 아닌 리뷰 중심의 메인 페이지 구성. 예: “이번 주 인기 리뷰”, “피부MBTI별 추천 후기”, “신규 리뷰러 소개”
- 리뷰 썸네일 카드화: 리뷰를 이미지+요약 텍스트로 카드화하여 탐색을 쉽게. 예: 인스타그램 피드처럼 구성
- 리뷰 태그 클러스터링: “#보습강력”, “#민감피부극찬” 등 태그 기반으로 리뷰를 묶어 탐색 가능하게

- 리뷰 챌린지 운영: 예: “이번 달 #겨울보습템 리뷰 챌린지” → 참여자 중 베스트 리뷰 선정
- 리뷰러 레벨 시스템: 리뷰 작성 수, 좋아요 수, 채택 수 등에 따라 레벨 부여 (예: “피부 리뷰 마스터 Lv.3”)

3. 신뢰도 및 공감 요소 강화
- 피부MBTI별 리뷰 통계 제공: 예: “DSNT 사용자 중 87%가 만족” → 리뷰 신뢰도 상승

- 리뷰 공감 버튼 다양화: “나도 이랬어요”, “정보 유익해요”, “사진 도움돼요” 등 좋아요 외 공감 방식 추가

4. 리뷰 기반 추천 기능 강화
- 리뷰 기반 상품 추천: “당신과 피부MBTI가 같은 사람들이 많이 리뷰한 제품” 추천
- 리뷰 기반 검색어 자동완성: 예: “건조한 피부” 입력 시 → “건조한 피부에 좋았던 리뷰 많은 제품” 자동 제안

5. 리뷰 작성 유도 및 보상 강화
- 리뷰 미션 시스템: 예: “이번 주 3개 제품 리뷰하면 추가 포인트 지급”
- 리뷰 작성 가이드 제공: 예: “피부 상태, 사용 기간, 사용 방법, 추천 여부” 등 항목별 가이드

🧩 리뷰 중심 기능 강화 전략 (테이블 확장 없이)
1. 🔍 리뷰 탐색 기능 강화
- 리뷰 중심 검색 탭 구성: 상품 검색과 별도로 리뷰 전용 탭 제공 (전체/베스트/피부MBTI별/태그별)
- 정렬 옵션 다양화: 좋아요 순, 최신순, 피부MBTI 일치율 순 등
- 피부MBTI 필터링: 사용자와 동일한 피부MBTI를 가진 리뷰만 보기
- 태그 기반 탐색: 기능성 태그(#보습강력 등)로 리뷰 분류 및 탐색

2. 🖼️ 리뷰 콘텐츠 다양화
- 이미지 리뷰 강조: 리뷰 썸네일에 이미지 표시 → 리뷰 탐색 시 시각적 정보 제공
- 리뷰 요약 키워드: 기능성 태그 기반으로 리뷰 요약 키워드 자동 생성
- 리뷰 댓글 활성화: 댓글창에 구매 여부 표시 (비구매자는 힌트 텍스트로 구분)

3. 🧠 리뷰 신뢰도 및 공감 기능 강화
- 피부MBTI별 만족도 통계: 특정 제품에 대해 피부MBTI별 평점/리뷰 수 제공
- 공감 버튼 다양화: 좋아요 외에 “나도 이랬어요”, “정보 유익해요” 등 공감 방식 추가
- 운영자 채택 리뷰 강조: 별도 표시 및 메인 노출 (is_selected 활용)

4. 🎯 리뷰 기반 추천 기능
- 피부MBTI 기반 상품 추천: 사용자 피부MBTI와 일치하는 제품 우선 추천
- 리뷰 많은 상품 우선 노출: review_count 높은 상품을 추천 리스트 상단에 배치
- 리뷰 기반 자동완성 검색어: 리뷰 키워드 기반으로 검색어 제안

5. 🎁 리뷰 보상 시스템 자동화
- 좋아요 수 기준 상위 리뷰 보상: 월별 좋아요 상위 10% 리뷰에 추가 혜택 지급
- 중복 지급 방지: 포인트 지급 이력에 리뷰 ID 기록하여 중복 방지
- 최소 좋아요 수 기준 설정: 예: 100개 이상 좋아요 받은 리뷰만 보상 대상

6. ✍️ 리뷰 작성 유도
- 리뷰 미션 시스템: “이번 주 3개 리뷰 작성 시 500포인트 지급” 등 미션 제공
- 리뷰 작성 템플릿 제공: 피부 상태, 사용 기간, 사용 방법 등 항목별 가이드
- 리뷰 작성 후 즉시 피드백: “리뷰가 등록되었어요! 3명이 공감했어요” 등 실시간 반응

7. 🛠️ 운영자 기능 강화
- 운영자 리뷰 채택 기능: 관리자 페이지에서 리뷰 선택 가능 (is_selected)
- 기능성 태그 부여: 리뷰에 태그 추가 가능 (REVIEW_TAG 활용)
- 리뷰 모니터링 시스템: is_checked 필드 활용하여 운영자 검토 여부 관리
